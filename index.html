    Project Title: Personal Cybersecurity Awareness Guide

    This semester-long project focuses on creating a professional cybersecurity awareness website to serve as both a learning tool for the public and a portfolio piece that demonstrates my skills to potential employers. The website will include clear, beginner-friendly lessons on online safety, phishing, and password management, along with interactive examples and a quiz to test user knowledge. Throughout the project, I will research current cybersecurity best practices, design engaging educational content, and develop a fully functional and visually appealing website. This project will also act as a career development tool by showcasing my ability to explain technical concepts clearly, design accessible user interfaces, and apply cybersecurity knowledge in a practical, real-world context. Completing and publishing this website will strengthen my professional portfolio and help me attract interviews for internships or entry-level roles in cybersecurity and IT.
